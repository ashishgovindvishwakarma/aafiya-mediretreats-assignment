# Aafiya Mediretreats – Total Knee Replacement Landing Page

A responsive medical treatment landing page developed as part of the Ideamagix Review Test Assignment.

This project recreates the provided design reference using HTML5, CSS3, JavaScript, jQuery, Bootstrap, Font Awesome, and Owl Carousel. It is built as a static frontend webpage and designed to closely match the supplied desktop and mobile layouts.

## Project Objective

The objective of this assignment was to develop a responsive webpage exactly similar to the provided design reference while following the required instructions:

- Use HTML5 tags
- Make the webpage responsive
- Add lazy loading to images
- Use Owl Carousel for sliders
- Ensure cross-browser support
- Use JavaScript or jQuery for the mobile menu
- Add a full-screen page loader that fades out after load

## Features Implemented

- Semantic HTML5 structure
- Responsive layout for desktop and mobile
- Lazy loading on relevant images
- Full-screen page loader with fade-out effect
- Mobile menu using jQuery
- Sticky anchor tabs
- Active section tab highlighting on scroll
- FAQ accordion
- Language dropdown
- Fade-up scroll animations
- Owl Carousel sliders for patient stories, doctors, and hospitals
- Footer call-to-action section
- Multi-column footer layout
