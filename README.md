# Aafiya Mediretreats – Total Knee Replacement Landing Page

This project is a responsive medical treatment landing page developed as part of the **Ideamagix Review Test Assignment**.

It is built using **HTML5, CSS3, JavaScript, jQuery, Bootstrap, Font Awesome, and Owl Carousel** and follows the provided design reference as closely as possible.

## Assignment Objective

The objective of this assignment was to recreate the provided webpage design using the required frontend technologies while ensuring responsiveness, proper UI interactions, and browser compatibility.

---

## Features Implemented

- Semantic **HTML5** structure
- Responsive **desktop and mobile** layout
- **Lazy loading** on images where applicable
- **Page loader** with fade-out effect
- **Mobile navigation menu** using jQuery
- **Sticky anchor tabs**
- **FAQ accordion**
- **Language selection dropdown**
- **Scroll reveal animations**
- **Owl Carousel** sliders for:
  - Patient Stories
  - Top Doctors
  - Recommended Hospitals
- Footer CTA and multi-column footer section
- Compatible with modern browsers
